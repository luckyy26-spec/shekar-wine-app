# Shekar - Mood-Based Wine Concoctions

Create personalized elixirs based on mood, flavor, and alcohol level.

Built with:
- React + Vite
- TailwindCSS + ShadCN
- Radix UI
- TypeScript

Features:
- Flavor compatibility checker
- Mood-based alcohol recommendations
- GCash payment integration
- Charity support

To run:
```bash
npm install
npm run dev
